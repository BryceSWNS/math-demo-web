## 字段填写规则

### 1) 标题（必填）

- 建议：简洁描述题目主题，例如“二次函数与面积最值”

### 2) 身份与昵称（必填）

- 身份：学生
- 昵称：D学长

### 3) 题干 `stemMd`（必填）

- 支持：`Markdown + LaTeX`
- 排版建议：希望题干更紧凑，优先使用行内公式 `$...$`，仅在需要单独居中展示时再用 `$$...$$`

### 4) 选项 `optionsRaw`（可选）

- 按行填写，每行一个选项
- 推荐格式：
  ```text
  A. 1/3
  B. 1/2
  C. 1
  D. 2
  ```
- 规则说明：
  - 若使用 `A.` / `B.` / `C.` 前缀，会按前缀作为选项键
  - 若不写前缀，系统会自动按 `A/B/C/...` 编号
  - 选项内容支持 LaTeX；推荐写法：`A. 根为 $2$ 和 $3$，且 $x_1^2+x_2^2=13$`
  - 系统会尝试自动识别如 `x1^2+x2^2=13` 这类常见表达式并渲染，但复杂公式仍建议显式使用 `$...$`

### 5) 答案 `answerMd`（可选）

- 支持：`Markdown + LaTeX`
- 用于给出最终结论或关键步骤

### 6) 解析 `analysisMd`（可选）

- 支持：`Markdown + LaTeX`
- 建议分步骤说明推导过程，便于学生阅读

### 7) 标签 `tagsRaw`（可选）

- 仅区分 微观/统计

### 8) 难度 `difficulty`（必选）

- 可选值：`easy` / `medium` / `hard`
- 页面上显示为：简单 / 中等 / 困难

### 9) 附件（暂时留空，不填）

- 支持类型：图片、PDF
